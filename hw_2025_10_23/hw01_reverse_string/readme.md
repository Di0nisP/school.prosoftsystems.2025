Эта программа считывает строку из стандартного ввода, удаляет хвост ввода до перевода строки (если пользователь ввёл больше символов, чем допускается), разворачивает строку и выводит результат.

### Особенности

- Ввод ограничен string_sz_max (по умолчанию 10 символов).
- Если пользователь ввёл больше - лишние данные читаются и сбрасываются (trim), чтобы не засоряли stdin.
- Строка разворачивается с помощью внешней функции reverse_string, подключаемой через io.inc.
- Используются системные вызовы Linux (read, write, exit).

| Файл               | Описание                                                              |
| ------------------ | --------------------------------------------------------------------- |
| `main.asm`         | Основной код программы: ввод, обрезка, вызов `reverse_string`, вывод. |
| `../shared/io.inc` | Заголовок с объявлениями внешних функций.                             |
| `../shared/io.asm` | Реализация вспомогательных функций (в т.ч. `reverse_string`).         |
| `Makefile`         | Сборка, запуск, генерация отладочного вывода `strace`.                |

### Сборка и запуск

Цель по-умолчанию: сборка + запуск

```
make run
```

#### Пример выполнения

- Если ввести больше 10 символов.

```
Enter your string:
1234567890HELLO
Reversed string:
987654321
```

Один из символов - `'\n'`
